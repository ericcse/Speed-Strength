2 days / week  •  Sprints → Jumps → Lift  •  Same playbook, your own loads
Day 1 — Horizontal

Acceleration + Broad Jumps + Lift

Hip / hamstring / glute focus. Sprints get off the line; broad jumps push forward.
Day 2 — Vertical

Max Velocity + Box/Pogo Jumps + Lift

Top-end speed & bounce. Jumps go up; adds arm work.
Every session, in order: Warm-up → Sprints → Jumps → Lift. Speed & power always come before fatigue — don't flip it. Progression (lift only): build sets first (2→3→4), then hold 4 sets and add reps.

Day 1 is the highest-strain session. Sprints + broad jumps + RDL all hammer the same hamstrings/glutes — the #1 sprint injury site. Warm up thoroughly, keep the RDL light, and start jumps near 10–15 (not 30). Rest ~1.5–2 min between lift sets; near-full recovery between sprint/jump reps.
Progression Path main lifts only — not biceps / triceps / core
Phase A · Build Sets
2×5Week 1›3×5Week 2›4×5Week 3
Phase B · Add Reps
4×6+1 rep›4×7+1 rep›4×8keep going
Tip-off at 5–6 reps. Hold each bubble ~1 week; advance when every rep is crisp. Sprints & jumps aren't scored this way — they grow by distance & clean landings.
